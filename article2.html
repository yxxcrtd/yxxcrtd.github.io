JDK

一，Windows 下安装 JDK

JAVA_HOME D:\JDK8

CLASSPATH .;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;

PATH %JAVA_HOME%\bin;


二，Linux 下安装 JDK

1，检查：rpm -qa | grep java (or rpm -qa | grep jdk)

2，删除：yum -y remove java-*

3，下载：http://www.oracle.com/technetwork/java/javase/downloads/index.html（历史版本：http://www.oracle.com/technetwork/java/archive-139210.html）

4，解压：tar zxvf jdk-*

5，移动：mv jdk1.8.0_101/ /home/jdk

6，配置：vim /etc/profile，追加：

export JAVA_HOME=/home/jdk

export PATH=$PATH:$JAVA_HOME/bin

export CLASSPATH=.:$JAVA_HOME/jre/lib/rt.jar:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar

7，生效：source /etc/profile

8，验证：java -version 或 javac -version


三，Mac 下安装 JDK

1，安装 java8 / java11：brew install java8 / java11

2，安装最新版本：brew install java

3，设置环境变量：vim .bash_profile 或 vim .zshrc

export JAVA_HOME=/Library/Java/JavaVirtualMachines/jdk1.8.0_101.jdk/Contents/Home

export CLASSPATH=".:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar"

export PATH="$JAVA_HOME/bin:$PATH"


四，Mac 下安装 OpenJDK

1，安装最新版本：brew install java

2，安装路径：/usr/local/opt/openjdk@11，实际位置：/usr/local/Cellar/openjdk@11

3，软连接到 JavaVirtualMachines 后配置环境变量：

sudo ln -sfn /usr/local/opt/openjdk@11/libexec/openjdk.jdk /Library/Java/JavaVirtualMachines/openjdk-11.jdk
