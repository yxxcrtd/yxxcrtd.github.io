jQuery 之 FancyBox 插件

一，前提

1，下载最新 jQuery：https://jquery.com/download/

2，下载最新的 FancyBox：https://fancyapps.com/fancybox/3/


二，使用

1，添加 css

<link rel="stylesheet" type="text/css" href="css/fancybox.3.5.7.css" />

2，添加 js

<script src="js/jquery-3.6.0.js"></script>

<script src="js/jquery.fancybox.3.5.7.js"></script>


三，图片应用

1，在图片的 a 标签上加上：data-fancybox

<a href="images/article/article34.01.png" data-fancybox>

    <img src="images/article/article34.01.png" width="835" />

</a>
