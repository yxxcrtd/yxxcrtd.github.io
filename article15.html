About GitHub

一，解决 GitHub 授权的问题

1，检查 SSH keys，如果之前有的话，最好都删除，再将本地的 id_rsa.pub 重新上传上去

2，测试：

ssh -T git@github.com

出现：Hi 用户名! You've successfully authenticated, but GitHub does not provide shell access. 表示成功了！

3，现在使用 SourceTree 测试

4，或者使用命令：

git clone git@github.com: 用户名/******.git


二，Please make sure you have the correct access rights and the repository exists.

1，删除 .ssh 文件夹

2，重新生成密钥和公钥：

ssh-keygen -t rsa -C "邮箱"，全部回车

ssh-keygen

3，拷贝 id_rsa.pub 内容到 GitHub 的 SSH keys

4，测试：

ssh -T git@github.com
