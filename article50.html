Visual Studio Code 下的 Java 开发

“ VS Code 的全称是 Visual Studio Code，是一款开源的、免费的、跨平台的、高性能的、轻量级的代码编辑器。

一，Mac 下安装方法

brew install visual-studio-code


二，常用设置

1，隐藏某些文件：Settings -> 输入：file exclude -> 当个文件是：.idea；文件夹是：**/out

2，禁用/启用小地图：Settings -> Text Editor -> Minimap -> Enabled 或 菜单栏 view -> Show Minimap

3，设置微调与建议：Settings -> Text Editor -> Suggestions

4，遥测技术影响性能：Settings -> Preferences -> Text Editor -> Telemetry

5，禁用/启用代码透镜：Settings -> Preferences -> Text Editor -> Code lens

6，查看进程和内存：Shift + Command + P -> developer: open process explorer


三，安装 VS Code 插件

1，Java Extension Pack

2，Spring Boot Extension Pack

3，Lombok

4，Gradle Extension Pack


四，macOS 下的常用快捷键

1，打开文件：command + p

2，在当前文件中搜索：command + g

3，全局搜索：shift + command + f

4，格式化代码：shift + option + f

5，格式化 import：shift + option + o

6，查看方法被使用位置：选中方法 + command + d

7，查看接口的实现：fn + command + F12

8，删除整行：Command + X

9，多个错误或警告之间逐个跳转：fn + F8

10，查看方法的所有引用：fn + shift + F12

11，变量重构：fn + F2

12，缩放：command + -/+

六，Maven 项目更新（代码更新、jar包导入报错等）

1，Shift + Command + P -> Maven: Execute Commands... -> 回车 -> 项目名称 -> 回车 -> clean

2，Shift + Command + P -> Maven: Execute Commands... -> 回车 -> 项目名称 -> 回车 -> compile

3，或者终端运行：

mvnd clean compile -Dquickly


七，Gradle 项目更新（代码更新、jar包导入报错等）

方法1，在 JAVA PROJECTS 后面点3个点中找：Clean Workspace -> Restart and delete

方法2，Shift + Command + P -> Java: Clean Java Language Server Workspace（推荐）
