Java 监控工具 JVisualVM 或 JConsole

“ JDK 自带的 JVisualVM 监控工具能有效的帮助我们监控 Java 应用程序运行过程中占用消耗的CPU、内存、类和线程等资源信息。

一，java -jar 方式配置

java -Dcom.sun.management.jmxremote -Dcom.sun.management.jmxremote.port=6666 -Djava.rmi.server.hostname=192.168.1.2 -Dcom.sun.management.jmxremote.ssl=false -Dcom.sun.management.jmxremote.authenticate=false -jar foo.jar


二，本地终端启动：jvisualvm

1，远程 -> 右键：添加远程主机 -> 填写主机名即可：192.168.1.2

2，192.168.1.2 -> 右键：添加 JMX 连接 -> 在 连接 中补充端口：192.168.1.2:6666


三，本地终端启动：jconsole
