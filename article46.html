Jar 包部署到 Docker

“ 直接将 jar 包部署到 Docker 中并运行

1，将 jar 包放到指定目录

2，在 jar 包同级目录创建 Dockerfile，vim Dockfile
FROM java:8
MAINTAINER yxx
ADD tams.jar demo.jar
EXPOSE 8080
ENTRYPOINT ["java", "-jar", "demo.jar"]
RUN /bin/cp /usr/share/zoneinfo/Asia/Shanghai /etc/localtime
RUN echo "Asia/Shanghai" >/etc/timezone

3，构建镜像：docker build -t tams .（注意：最后有一个点；tams 是构建之后的镜像名称，可以加版本号）

4，运行镜像：docker run -e TZ="Asia/Shanghai" -d --restart=always --name tams -p 8081:8080 project/tams

（或者：：docker run --name tams -p 8081:8080 tams）

5，查看日志：docker logs --tail 300 -f tams
