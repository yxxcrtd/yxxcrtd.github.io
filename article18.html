Spring Cloud Alibaba

一，Spring Cloud、Spring Cloud Alibaba 和 Spring Boot 三者版本对应关系

1，参考地址：https://github.com/alibaba/spring-cloud-alibaba/wiki/版本说明


二，Docker 下安装：Nacos

1，Nacos 官网：https://nacos.io/zh-cn/

2，下载与启动：

docker pull nacos/nacos-server

docker run --env MODE=standalone --name nacos -dp 8848:8848 nacos/nacos-server

3，访问：http://localhost:8848/nacos
