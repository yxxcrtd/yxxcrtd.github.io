Gradle 基础

一，下载

1，到 Gradle 的 releases 页面：https://gradle.org/releases/

2，下载：binary-only 或 complete（包含 docs 和 sources）

3，Mac 下配置环境变量：vim .zshrc

export GRADLE_HOME=/Users/young/gradle-6.7.1

export GRADLE_USER_HOME=/Users/young/repository

export PATH=$PATH:$GRADLE_HOME/bin

4，Windows 下配置环境变量：

GRADLE_HOME D:\gradle-6.7.1

GRADLE_USER_HOME D:\repo

PATH %GRADLE_HOME%\bin

5，验证：gradle -v

6，单模块打包：

gradle clean build

7，多模块打包（进入主Main的模块）：

gradle clean bootJar
