Git 基础

一，Linux CentOS 下安装 Git

1，安装依赖：yum -y install gcc perl-ExtUtils-MakeMaker curl-devel expat-devel gettext-devel openssl-devel zlib-devel

2，下载：wget https://www.kernel.org/pub/software/scm/git/git-2.10.1.tar.gz

3，解压：tar zxvf git-2.10.1.tar.gz

4，进入：cd git-2.10.1

5，编译：make prefix=/home/Git all

6，安装：make prefix=/home/Git install

7，添加环境变量：echo "export PATH=$PATH:/home/Git/bin" >> /etc/bashrc（在此文件上加一个：:/root/bin:/home/Git/bin）

8，使环境变量生效：source /etc/bashrc

9，验证：git --version，出现：git version 2.10.1，表示安装成功！(/home/Git/bin/git)


二，Mac 下安装 Git

1，brew install git

2，验证：git --version，出现：git version 2.10.1，表示安装成功！


三，删除文件夹下的所有 .git 文件

find . -name ".git"| xargs rm -Rf


四，升级后无法git，/usr/bin/git

终端执行：xcode-select --install


五，在用户文件夹下 定义一个全局的 Git 忽略文件：.gitignore_global

*~

.DS_Store

_CodeGenerateController.java

src/main/webapp/WEB-INF/ftl/_code

......


六，用户文件夹下的 Git 配置文件：.gitconfig


七，用户文件夹下的 Git 配置文件：.gitflow_export

export FLAGS_GETOPT_CMD=/Applications/Sourcetree.app/Contents/Resources/bin/getopt




八，SourceTree 中项目加入 Git Flow

1，仓库初始化：点 SourceTree 中右上角的 Git Flow -> OK

2，再点右上角的 Git Flow -> Start a New Feature -> 填写 Feature 名称（Feature 只与 develop 分支交互）

3，Feature 完成后，先把 develop 合并到当前 Feature 分支（如果有冲突，要先解决冲突），再点右上角的 Git Flow -> 完成功能 -> 此时会合并到 develop（推荐勾选：删除当前 Feature 分支）


九，Git Flow 在 完成功能的时候出现错误：fatal: could not read Username for 'https://gitee.com': No such file or directory

1，进入命令行模式，执行：

git flow feature finish -k 功能名称


九，安装：SourceTree

1，下载 SourceTree：https://www.sourcetreeapp.com

2，安装 SourceTree：需要有 Atlassia 的账户

3，在本地生成公钥和私钥：打开 SourceTree 右上角的终端：ssh-keygen，全部回车

ssh-keygen

4，将 cat ~/.ssh/id_rsa.pub 中的内容拷贝到 GitLab 的 Profile Settings -> SSH Keys 中，点一下 Title，保存

cat ~/.ssh/id_rsa.pub

5，授权（1，输入：yes，出现：Welcome to GitLab, @young!，说明你已经有访问 GitLab 的权限；2，默认的端口参数-p 22可以不加）

ssh -T -p 22 git@192.168.1.8

6，如果 GitLab 服务器上的 22 端口不能使用，则需要在本地的 .ssh 文件中添加一个 config 文件，内容如下：（上面的授权需要添加端口参数）

Host 192.168.1.8

Hostname 192.168.1.8

Port 10222

7，在 SourceTree 中配置连接 GitLab，下载代码


十，Mac 下安装 SourceTree

brew cask install sourcetree


十一，，添加本地已有项目到 GitHub

1，首先在 GitHub 上创建一个项目：https://github.com/yxxcrtd/demo（demo就是你创建的项目名称）

2，打开 SourceTree -> 项目列表 -> 新建 -> 添加已经存在的本地仓库 -> 选择已有的仓库文件夹 -> 打开

3，双击 demo -> 右上角 -> 设置 -> 远程仓库 -> 添加 -> 远程（仓库）名称：origin -> URL/路径：git@github.com:yxxcrtd/demo.git -> 确定

5，提交代码
