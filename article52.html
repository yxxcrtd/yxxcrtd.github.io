Redis 集群

“ Redis 集群是 Redis 提供的分布式 数据库 方案，集群通过分片 (sharding)来实现数据共享，并提供复制和故障转移。

一，环境搭建

1，Redis 集群至少需要3个主节点，3个从节点。生产环境推荐使用6台服务器。

2，安装参考：Redis 基础

3，编辑6个配置文件：vim redis.conf，修改3个配置即可

#bind 127.0.0.1

protected-mode no

cluster-enabled yes

4，启动6个redis：

/home/redis/src/redis-server /home/redis/redis.conf &

5，至此6个节点没有关联，创建集群：

redis-cli --cluster create --cluster-replicas 1 192.168.90.82:6379 192.168.90.83:6379 192.168.90.84:6379 192.168.90.219:6379 192.168.70.207:6379 192.168.70.208:6379

如果有密码，可以通过-a传参
