MinIO 基础

“ MinIO 是一款高性能、分布式的对象存储系统，提供高性能、与 S3 兼容的对象存储，让你自己能够构建自己的云储存服务，它适用于AI的高性能对象存储，是世界上最快的对象存储，没有之一。

一，前置：

1，官网：https://min.io

2，由于最新版本上传文件后，资源无法展示，所以选择了低版本的 Docker Images：minio/minio:RELEASE.2021-06-17T00-10-46Z

3，由于 Docker Compose 无法启动：RELEASE.2021-06-17T00-10-46Z，所以只能使用Docker命令安装


二，Docker Compose 安装 MinIO （适用于最新版本）

version: '3.1'

services:

minio:

      image: 'minio/minio'

      container_name: minio

      restart: always

      ports:

          - "9000:9000"

          - "9001:9001"

      environment:

          MINIO_ROOT_USER: minio

          MINIO_ROOT_PASSWORD: 123456

      volumes:

          - /home/minio/data:/data

          - /home/minio/config:/root/.minio/

      command: server /data --console-address ':9001'

      privileged: true

启动：

docker-compose up


三，Docker 安装 MinIO（指定版本：minio/minio:RELEASE.2021-06-17T00-10-46Z）

docker run --name minio -d --restart=always -p 9000:9000 -p 9001:9001 -v /home/minio/data:/data -e MINIO_ROOT_USER="minio" -e MINIO_ROOT_PASSWORD="123456" minio/minio:RELEASE.2021-06-17T00-10-46Z server /data


四，访问与配置

1，访问：http://127.0.0.1:9000（minio/123456）

2，创建一个存储桶：bucket1

3，设置存储桶的权限，增加一个 Bucket Policy (bucket1)：Read and Write
