Docker 私有镜像仓库 - Registry & Harbor

一，创建需要的文件目录：

mkdir /home/docker/registry -pv -m 777


二，启动：

docker run --detach --restart always --name registry --publish 5000:5000 --volume /home/docker/registry:/var/lib/registry --volume /usr/share/zoneinfo/Asia/Shanghai:/etc/localtime registry


三，增加防火墙端口：

firewall-cmd --zone=public --add-port=5000/tcp --permanent


四，安装客户端：

docker run --name dockerfly -d -v /var/run/docker.sock:/var/run/docker.sock --restart always -p 2735:2735 -p 28083:28083 helyho/dockerfly


五，访问：

http://192.168.1.2:28083（admin/1234）
