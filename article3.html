树莓派

“ Raspberry Pi（中文名为“树莓派”，简写为：RPi），只有信用卡大小的微型电脑，系统基于 Linux。

一，准备

1，购买：树莓派，官网、中文站

2，购买：TF 卡（需要速度快的 A1、A2）

3，下载（修改地址中相应的版本号，或者到：http://mirrors.huaweicloud.com/centos-altarch/）：

CentOS 7：https://mirrors.tuna.tsinghua.edu.cn/centos-altarch/7.9.2009/isos/armhfp/CentOS-Userland-7-armv7hl-RaspberryPI-GNOME-2009-sda.raw.xz

CentOS 8：http://mirrors.huaweicloud.com/centos-altarch/8.1.1911/isos/armhfp/CentOS-Userland-8-armv7hl-generic-GNOME-1911-sda.raw.xz

1，注意 CentOS 7 和 CentOS 8 的版本差别，7是：RaspberryPI ；8是：generic，暂不推荐

2，继续关注 CentOS 8 的 RaspberryPI 版本：https://mirrors.tuna.tsinghua.edu.cn/centos-altarch/8.1.1911/isos/armhfp/

3，推荐安装 GNOME 版本，不推荐 mini 版本

4，安装写卡工具：

brew install balenaetcher

5，装卡 -> 接电 -> 开机

6，树莓派各版本参数对比：（3代b型E14中国版）


二，Mac 下使用 balenaEtcher 写 256G 的 TF 卡会出现错误：/dev/rdisk2: Operation not permitted

1，查看磁盘列表：

diskutil list

2，擦除TF卡的分区保护：

sudo diskutil partitionDisk /dev/disk2 1 MBR "Free Space" "%noformat" 100%

确定 disk2 是 TF卡的分区

3，退出TF卡：

sudo diskutil eject /dev/rdisk2

4，再次使用 balenaEtcher 工具写卡


三，使用 Mac 格式化 TF 卡后，容量变小的恢复

1，查看磁盘列表：

diskutil list

2，格式化为 FAT32 格式：

sudo diskutil eraseDisk FAT32 RASPBIAN MBRFormat /dev/disk2

RASPBIAN：名称，可以自定义，不能包含小写字母

disk2 是第一步查出来的磁盘名称，千万保证正确！

3，退出TF卡：

sudo diskutil eject /dev/rdisk2
