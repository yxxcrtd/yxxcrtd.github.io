IDEA 远程调试

一，在本地配置 IDEA

1，菜单栏 Run -> Edit Configurations

2，添加一个：Remote

3，修改 IP 地址、端口号可以用默认的 5005

4，拷贝命令行参数：-agentlib:jdwp=transport=dt_socket,server=y,suspend=n,address=5005


二，在服务端启动：远程服务

java -jar -agentlib:jdwp=transport=dt_socket,server=y,suspend=n,address=5005 /home/test.jar


三，在本地启动 IDEA

1，点 远程的 Unnamed 的 调试 按钮，控制台只输出一行：（如果有错误，在左下角会有错误提示）

Connected to the target VM, address: '192.168.3.203:5005', transport: 'socket'


四，本地浏览器访问

http://192.168.3.203:5005

1，如果服务端的 jar 包有端口，则继续用之前的端口

2，没有的话，则用 5005


五，Tomcat 7.0.70 配置远程调试

1，vim /home/apache-tomcat-7.0.70/bin/catalina.sh

2，添加：CATALINA_OPTS="-server -Xdebug -Xnoagent -Djava.compiler=NONE -Xrunjdwp:transport=dt_socket,server=y,suspend=n,address=5005"
